1. Introduction

The Slim Mobile UI Kit is a lightweight CSS framework designed for building mobile-friendly web applications. It provides a set of essential components and utilities with a clean, modern look that's suitable for mobile interfaces. The kit also includes support for Right-to-Left (RTL) layouts, making it versatile for global applications.

2. Getting Started

To use the Slim Mobile UI Kit in your project:

  1. Download the `slim-mobile-ui.css` file.
  2. Link the CSS file in your HTML: ```html ```
  3. Start using the classes in your HTML markup.

4. Utility Classes

The kit includes several utility classes for common styling needs:

        - `.text-center`: Center-align text
        - `.mt-1`, `.mt-2`: Add top margin
        - `.mb-1`, `.mb-2`: Add bottom margin
        - `.ml-1`, `.ml-2`, `.mr-1`, `.mr-2`: Add left or right margin (with RTL support)

6. RTL Support

To enable RTL layout, add the `dir="rtl"` attribute to your `` or ` ` tag:

The kit automatically adjusts layouts, text alignment, and margins for RTL languages.

7. Customization

To customize the UI kit, modify the CSS variables at the top of the `slim-mobile-ui.css` file. This allows you to easily change colors, fonts, and other properties throughout the kit.

8. Browser Support

The Slim Mobile UI Kit is designed to work with modern browsers that support CSS3. It has been tested on the latest versions of Chrome, Firefox, Safari, and Edge.

For more detailed examples and advanced usage, please refer to the source code comments in the `slim-mobile-ui.css` file.
